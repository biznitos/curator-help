UPLOADER
--------

Implement a file uploading service

Definining
----------
You can define an uploader by creating a template as follows:

permalink format:
uploader-[slug]

Example:
uploader-cleaning

Formatting
----------
Each line represents a file that is required to be uploaded in this format:

title|filename|accept-mime-type|example-file-url

Example:
Upload your passport|filename|image/*,application/pdf|https://upload.wikimedia.org/wikipedia/commons/0/0d/Biodata_page_of_Singapore_Passport.jpg

Notes
-----
THe examile-file-url can be blank

The title of the template will be shown on the upload screen.
