SIGNUP

Endpoint
--------
/api/singup/v1

Verb
----
POST

Generic Parameters
(Universal to all sites)
------------------
* email (required)
* name
* phone
* gender
* first_name
* last_name
* middle_name
* occupation
* street
* city
* country_code
* dob (date of birth)
* url
* meta1
* meta2
* meta3

Site Parameters
(For specific sites)
---------------
* data (map of many parameters)
* utm_xxxx variables

Control Parameters
------------------
* return_url (where to go after successful signup)
* failure_url (where to go when signup failed)
* login (if set to any value, then will also login user after successful signup)
